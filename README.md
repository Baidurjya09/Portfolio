## 🚀 Portfolio Website

A modern, cyberpunk-themed portfolio showcasing AI/ML projects, full-stack development work, and professional experience.

### ✨ Features

- **Hero Section** - Animated introduction with parallax effects
- **Profile** - About me with statistics
- **Arsenal** - Technical skills & tools (AI/ML, Web Dev, Languages, Databases)
- **Projects** - 4 major projects with 3D card stacking effect:
  - CivicMind (Multi-agent civic governance simulation)
  - GORMODE-A (Emergency response AI with RAG)
  - AI Hiring Assistant (OpenEnv-compliant recruitment platform)
  - Multi-Disease Detection (ML-based health prediction)
- **GitHub Section** - Open source contributions & repositories
- **Resume** - Download/request CV
- **Experience** - Work history & internships
- **Contact** - Get in touch section

## 🌐 View Portfolio

Simply open `portfolio-enhanced.html` in any modern browser.

### Files

- `portfolio-enhanced.html` - Main portfolio (recommended)
- `portfolio.html` - Alternative version

## 📝 Customization

To customize with your own information, edit `portfolio-enhanced.html`:

1. **Personal Info** - Update name, title, bio in hero section
2. **Contact** - Update email, GitHub, LinkedIn links
3. **Projects** - Replace with your projects
4. **Experience** - Add your work history
5. **Skills** - Update Arsenal section with your skills
6. **GitHub** - Update repositories and stats

### Key Sections to Edit

- Line 3-74: SEO metadata & Schema.org data
- Line 532: Hero section (name, title, bio)
- Line 647: Projects
- Line 892: GitHub repositories
- Line 1029: Resume section
- Line 1082: Experience/logs
- Line 1138: Contact links

## 🚀 Deployment

### Option 1: GitHub Pages (Free)
1. Create a new repository
2. Upload `portfolio-enhanced.html` and rename to `index.html`
3. Enable GitHub Pages in Settings
4. Done!

### Option 2: Netlify (Free)
1. Drag & drop the `portfolio-enhanced.html` file
2. Rename to `index.html`
3. Deploy

### Option 3: Vercel (Free)
1. Import repository
2. Rename file to `index.html`
3. Deploy
